## 主要功能
- 直观操作界面：将输入的视频 URL 转化为对应的 yt-dlp 命令，无需手- 动输入复杂参数。
- 灵活参数设置：
    - 指定下载目录。
    - 选择下载单音频或音视频。
    - 配置代理服务器。
    - 是否转码为 MP4（需要 ffmpeg 支持）。
    - 插入视频封面、元信息及字幕。
- 命令预览与执行：自动生成 yt-dlp 命令，支持一键复制或在程序内直接执行，实时查看 Terminal 输出。
- 多主题支持：更换界面主题，提升用户体验。
